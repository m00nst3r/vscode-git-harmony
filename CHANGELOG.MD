# Changelog

## 0.2.0
### Added
* Decline button (✕) for each conflict block.
* Ability to toggle selections by declining the currently accepted side.
* Improved button layout placing accept actions closer to the result column.
* Visual indicators showing which side of a conflict was accepted by disabling buttons.

## 0.1.1

### Added

* App icon (`icon.png`) and `license` field in `package.json`

## 0.1.0

### Added

* Initial release with core features:
  - Conflict file list panel with bulk resolve actions
  - Full 3-column merge editor with per-block and manual resolution
  - Synchronized scrolling and syntax highlighting in the merge editor
  - Robust conflict marker parsing and Git CLI integration
